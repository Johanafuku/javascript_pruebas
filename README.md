Ejercicios usando Javascript: 
--
1. Cambiador de color: al hacer clic el fondo cambiar de color de forma aleatoria.
2. Contador de clics: al hacer clic en el botón "Contardor" va mostrando la cantidad de clics.
3. Lista dinámica: Dada una lista al hacer clic en el botón "Agregar" se añade el elemento a la lista, y "Eliminar" lo quita de la lista.
4. Filtro búsqueda en tiempo real: Dada una lista, cuando el usuario ingresa el texto se va filtrando dejando las palabras que hacen match.
5. Calculadora sencilla: Con 4 botones por cada operación de suma, resta, multiplicación y división, el usuario ingresa los 2 números.
6. Temporizador inicio, pausa, reinicio: Con los botones de "Iniciar", "Pausar", "Reiniciar" va contando segundos, minutos y horas, puedes pausar o reiniciar.
7. Generador de contraseñas aleatorias: El usuario ingresa la longitud de la contraseña y se genera combinando letras minúsculas, mayúsculas, números y caracteres especiales.
8. Contador de palabras y caracteres: Usuario escribe un párrafo y se va mostrando la cantidad de palabras y caracteres sin espacios en tiempo real.
9. Lista de tareas con localStorage: Uso de LocalStorage para guardar las tareas ya realizadas o eliminadas.

Exercises using Javascript:
--
1. Color change: By doing clic the background color changes in a random way.
2. Clic counter: by doing clic in the button "Contador" it shows the amount of clics.
3. Dinamic list: Given a list, you clic the button "Agregar" and adds the element to the list and "Eliminar" takes it out of the list.
4. Filter for real-time search: Given a list, when user enters a phrase or text code goes filtering leaving only the words that match.
5. Simple calculator: With 4 buttons for each operation of adding, substract, multiplication and division, user enters 2 numbers.
6. Timer with start, pause and restart: With the buttons of "Iniciar", "Pausar", "Reiniciar", it goes counting seconds, minutes and hours, you can pause or restart.
7. Random password generator: The user enters the length of the password and it is generated combining lower letters, capital letters, numbers and special chars.
8. Counter of words and characters: The user writes text and it goes showing the amount of words and characters without space in real-time.
9. To-do list with localStorage: Use of LocalStorage to save to-do tasks, marked completed or delete them.
